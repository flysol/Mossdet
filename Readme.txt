Installation instructions:

1.Copy these 3 files to the same folder
	- MOSDET.exe 
	- MCH_DP_FastRipples_5e-05_1e-05_12-9, 6_11_19.dat
	- SCH_UD_Ripples_0.00125_0.15625_12-1, 20_20_38.dat
	- SpikeDetector_PFSA80_NPR1_1e-05_1e-05_3-8, 5_27_31.dat

2.If the executable file fails to run on your computer, you need to install the run-time components of the Visual C++ libraries.
You will find the installer for Microsoft Visual C++ Redistributable(2017) under VC_redist.x64.exe
